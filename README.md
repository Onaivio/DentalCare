### Run Locally

To run **DentalCare** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/onaivio/DentalCare.git
```

Windows:

```bash
git clone https://github.com/onaivio/DentalCare.git
```
